# Titanic Data Preprocessing & Modeling 

## Project Overview
This project performs comprehensive data preprocessing and machine learning modeling on the Titanic dataset from Seaborn. The goal is to predict passenger survival based on various features through exploratory data analysis, cleaning, feature engineering, and model building.

---
## Project Structure 
- Exploratory Data Analysis (EDA)
- Data Cleaning
- Feature Engineering
- Categorical Encoding
- Outlier Detection & Handling
- Machine Learning Modeling
